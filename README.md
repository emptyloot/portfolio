### Prerequisites

  * Node.js 18.17 or later
  * npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone git clone https://github.com/emptyloot/portfolio.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

    ```bash
    npm run dev
    ```
